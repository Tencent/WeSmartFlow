# 联网检索

## 工具

| 工具 | 用途 |
|------|------|
| `tavily_search` | 通用网络搜索，返回 AI 提炼的高质量结果 |
| `arxiv_search` | 搜索学术论文，关键词用英文 |
| `web_fetch` | 抓取指定页面内容，配合搜索结果深入阅读 |

## 何时检索

- 知识图谱中不存在的概念，且无法从已有知识推导
- 时效性问题（"最新的"、"今年"等）
- 需要权威定义或具体数据支撑
- 用户提到具体的论文、技术文档

不适用：图谱中已有节点且信息足够、纯推理计算类问题。

## 流程

1. `tavily_search` 获取概览
2. 学术问题追加 `arxiv_search`（英文关键词更准）
3. 感兴趣的结果用 `web_fetch` 深入阅读
4. 回答时附上来源 URL
5. 有价值的新知识点 → `create_node` 纳入图谱
